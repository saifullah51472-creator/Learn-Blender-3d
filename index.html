💡 Tips & Shortcuts

Here are some of the most useful Blender tips and keyboard shortcuts to speed up your workflow:

🧭 Navigation

  • Middle Mouse Button (MMB) – Rotate view
  • Shift + MMB – Pan view
  • Scroll Wheel – Zoom in/out
  • NumPad 1/3/7 – Front/Side/Top view
  • NumPad 5 – Toggle Perspective/Orthographic

🛠 Modeling

  • Tab – Toggle Edit/Object Mode
  • A – Select/Deselect all
  • B – Box Select
  • C – Circle Select
  • E – Extrude
  • Ctrl + R – Loop Cut
  • G – Grab/Move
  • S – Scale
  • R – Rotate
  • Ctrl + Z – Undo | Shift + Ctrl + Z – Redo

✂️ Editing

  • X / Delete – Delete selected
  • F – Create Face
  • Ctrl + J – Join objects
  • P – Separate selection into new object
  • M – Move selection to new collection

🎞️ Animation

  • I – Insert Keyframe
  • Alt + I – Delete Keyframe
  • Shift + Left Arrow – Jump to first frame
  • Shift + Right Arrow – Jump to last frame
  • Spacebar – Play/Pause animation

🚀 Rendering & Misc

  • F12 – Render Image
  • Ctrl + F12 – Render Animation
  • Shift + A – Add new object
  • H – Hide selection | Alt + H – Unhide all
  • Ctrl + S – Save File

🌟 Pro Tips

  • Always use Layers/Collections to keep your scene organized.
  • Use Modifiers (Mirror, Subdivision, Boolean) for faster modeling.
  • Save frequently! Blender can crash sometimes.
  • Use MatCap shading (Viewport menu) for better modeling visibility.
  • Enable Auto Save & Incremental Save in Preferences.
